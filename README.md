# SmartStairLights

**SmartStairLights** — это проект автоматической подсветки лестницы с использованием адресной светодиодной ленты, управляемой ультразвуковыми датчиками. Проект позволяет включать и выключать подсветку в зависимости от направления движения человека по лестнице. Подсветка плавно загорается и гаснет, создавая эффект динамичной подсветки каждой ступени.

## О проекте

Цель проекта — создать "умную" подсветку для лестницы, которая автоматически включается, когда человек начинает подниматься или спускаться. Система основана на двух ультразвуковых датчиках (вверху и внизу лестницы), которые обнаруживают движение и активируют светодиодную ленту.

Основные возможности проекта:
- Динамическая подсветка: подсветка плавно включается с первой ступени по направлению движения.
- Контроль направления: система определяет, в какую сторону движется человек (вверх или вниз), и включает подсветку в соответствующем направлении.
- Энергосбережение: подсветка автоматически отключается через определённое время, если не зафиксировано движение.
- Поддержка разноцветной подсветки: возможно настроить различные цветовые эффекты, такие как радужная подсветка (Rainbow Effect).

## Аппаратные компоненты

Для реализации проекта использованы следующие компоненты:
- **NodeMCU (ESP8266)**: основной контроллер системы.
- **HC-SR04**: ультразвуковые датчики для определения движения.
- **Адресная светодиодная лента (WS2812B)**: светодиоды, которые управляются индивидуально и могут отображать различные цвета.
- **Блок питания 5V 70A**: для питания светодиодной ленты.
- **Резисторы, провода и коннекторы**: дополнительные элементы для подключения и защиты компонентов.

## Схема подключения

Система состоит из двух основных частей:
1. **Ультразвуковые датчики**: один датчик устанавливается в нижней части лестницы, другой — в верхней.
2. **Светодиодная лента**: она размещается вдоль лестницы и разделена на сегменты для каждой ступени.

Полную схему подключения можно найти в разделе "Схемы" (будет добавлено).

## Программное обеспечение

Проект написан на языке C++ с использованием **Arduino IDE**. Код реализует логику обнаружения движения, управления светодиодами и временные задержки для автоматического выключения подсветки.

### Основные функции:
- `detectMotion(Ultrasonic &sensor)`: функция для определения движения с помощью ультразвукового датчика.
- `animateStairs(bool bottomToTop)`: анимация включения светодиодов по направлению движения.
- `animateStairsOff(bool bottomToTop)`: анимация выключения светодиодов.
- Поддержка различных цветовых режимов: одноцветный режим, режим радужной подсветки и другие.
